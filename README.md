# Webpack 셋팅

## Project setup
```
npm install
```

### 개발서버 실행
```
npm run serve
```

### 배포용 빌드
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### 작업 내용
```
    - Vue-CLI 사용
    - 멀티 페이지 형식으로 사용 할 수 있도록 vue.config.js에서 옵션 재 설정
    - 각각 페이지당 생성되는 폴더에 HTML, CSS, JS 함께 생성
```

### 고려사항
```
    - 작업 진행전 프로젝트 폴더 구조 정의하여 사용
```
